# 🌤️ Application Météo Git

Une mini-application en ligne de commande développée en Node.js pour apprendre à maîtriser le flux de travail Git, GitHub et la gestion sécurisée des variables d'environnement.

## 🚀 Fonctionnalités
* Récupération de la météo en temps réel via l'API OpenWeatherMap.
* Gestion sécurisée de la clé d'API grâce à un fichier `.env` ignoré par Git.
